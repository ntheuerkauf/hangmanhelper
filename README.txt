Hangman Helper is a tool to help a person while playing the popular kids game Hangman

As the user finds out more information, such as confirmed or disconfirmed letters, they should enter that information into the program
Specifics on input can be found when "?" is entered
The most important inputs would be "S" and "P" because they print statistics on the most common letters remaining and the list of remainign words, respectivly

Future updates will clean up valid input detection and add functionality for words that don't have exactly five letters
